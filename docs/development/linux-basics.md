# Linux 入门

Linux 入门不需要一次掌握所有命令。先学会路径、文件、权限、进程和包管理器的基本概念。

## 起步命令

```bash
pwd
ls
cd
mkdir
cp
mv
rm
cat
grep
```

使用删除命令前确认路径，避免把练习命令直接套用到重要目录。
