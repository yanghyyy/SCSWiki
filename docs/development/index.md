# 开发环境总览

开发环境是反复使用的工具组合。建议从编辑器、终端、Git 和语言工具链开始，先追求可复现，再追求效率。

## 建议掌握

- 基本终端命令。
- Git 提交、分支和远程仓库概念。
- 编译器或解释器的安装检查。
- 错误信息的复制、搜索和归纳。
