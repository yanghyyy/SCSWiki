# C/C++ 入门

C/C++ 常用于理解程序运行、内存、编译和算法实现。本页不对应任何具体课程要求，只提供通用学习顺序。

## 学习顺序

1. 熟悉变量、分支、循环和函数。
2. 理解数组、字符串、结构体和指针的基本概念。
3. 学会用编译器查看报错信息。
4. 编写小程序并保存到 Git 仓库。
5. 用简单题目练习输入输出和边界条件。

## 常见误区

- 只复制代码，不读编译错误。
- 不保存可复现的最小示例。
- 在没有理解数组边界时处理复杂题目。
