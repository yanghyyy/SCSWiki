# 专业学习总览

本页只记录通用学习建议，不替代培养方案、课程大纲或任课教师要求。学分、先修关系、考试安排等信息必须以正式文件为准。

## 建议建立的基础

- 至少熟悉一门系统学习过的编程语言。
- 能使用 Git 管理个人代码。
- 理解基本命令行操作和文件路径。
- 逐步建立英文技术文档阅读能力。
