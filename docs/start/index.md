# 新生入门总览

本页暂不提供具体报到时间、地点、宿舍安排或学院通知内容。相关信息变化快，必须以学校或学院最新正式通知为准。

## 建议先做的事

- 阅读录取、报到和学院通知时，确认发布单位、发布日期和适用对象。
- 对群聊转发的信息保持谨慎，遇到冲突时优先查正式来源。
- 保存自己需要办理事项的原始通知链接或截图，但不要公开传播包含个人信息的材料。
- 熟悉本站的[内容可信度说明](/about/trust)。

## 待核验信息模板

如果维护者要补充新生指南，请先收集公开来源，再填写：

- 事项名称。
- 适用对象。
- 正式来源名称和 URL。
- 发布日期。
- 是否涉及个人信息或内部系统。
