# 新生常见问题

本页用于收集适合公开回答的通用问题。涉及具体时间、地点、名单、账号、流程和政策的问题，不在没有正式来源时给出答案。

## 可以先了解的问题

- 如何判断一个页面是否已经核验。
- 如何提交内容错误或过期反馈。
- 如何开始学习通用编程基础。

## 暂不直接回答的问题

- 报到当天的具体流程。
- 宿舍分配规则。
- 学院当前负责人员或联系方式。
- 任何账号、群号、电话或内部链接。
