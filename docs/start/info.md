# 学院信息快速一览

## 学院简介

计算机学院（School of Computer Science）目前开设本科生、硕士研究生招生，皆授予**工学学士、硕士**学位。

- 本科生专业（四年学制）：计算机科学与技术（师范）、软件工程、网络空间安全。
- 研究生专业（三年学制）：
  - 学术型研究生：软件工程
  - 专业型研究生：计算机技术、人工智能
