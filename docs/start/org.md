# 学院组织

学院目前通过**两委会**建立学生与老师之间的联系，具体分为**学生分会**和**团委秘书处**，由两委会与班级负责人选举出**主席团**成员。除去院级学生组织，一些**校级学生社团**也挂靠学院。

## 主席团

主席团成员全面统筹协调组织两委会和各班级负责人，进行学院活动的筹备、开展、执行。主席团成员主要分为两类：

- 学生分会主席（3人）
- 团委秘书处秘书长（1人）、团委秘书处副秘书长（1人）

## 学生分会

学生分会下设五大部门：

1. 纪检监察部
2. 综合事务部
3. 素质发展部
4. 学习发展部
5. 权益服务部

> 具体部门详情，请查看部门页面介绍。

## 团委秘书处

团委秘书处下设八大部门：

1. 创新创业部
2. 青年志愿者协会
3. 文艺部
4. 宣传部
5. 组织部
6. 新媒体中心
7. 大学生艺术团
8. 党建办（内部推荐）

> 具体部门详情，请查看部门页面介绍。

## 学生社团

目前有三个校级社团挂靠学院，分别是：程序设计算法协会、AIIC机器人社团、网安协会。同学们可根据自己喜好兴趣选择加入~

> 具体社团信息，请查看学生社团页面介绍。
