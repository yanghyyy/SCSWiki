# 使用说明

阅读页面时，先看页面顶部的内容可信度信息。它会标明内容类型、页面状态、最后更新、适用对象、维护者和来源情况。

## 页面状态

- `active`：当前可读，但仍需按页面来源判断适用性。
- `needs-review`：缺少可靠来源或等待维护者核验。
- `draft`：草稿，可能缺少结构或关键说明。
- `archived`：历史内容，不应直接作为当前依据。

## 处理不确定信息

如果页面涉及学校特定政策、流程或时间安排，但没有公开正式来源，请把它视为待核验信息。你可以通过 Issue 提交来源材料或建议修改。
