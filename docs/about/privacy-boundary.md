# 隐私与内容边界

本站不收集个人数据，不添加统计脚本，不使用 Cookie。贡献内容时也不要公开填写手机号、学号、群号、个人邮箱、内部系统链接或其他不适合公开的信息。

## 不接受的内容

- 教师或学生个人评价、排名和联系方式。
- 需要登录才能访问的内部资料。
- 未授权转载的课件、试卷、作业答案或教材内容。
- 可能误导读者的学校特定流程和时间安排。
