# 项目介绍

SCSWiki 是一个学生维护的非官方知识库，目标是把可复用的学习经验、工具入门、内容模板和贡献规范整理到一个可搜索、可审阅、可持续维护的地方。

本站不代表学校或学院立场。任何涉及政策、流程、时间、人员、联系方式、课程规则或奖助认定的信息，都必须回到学校或学院公开正式通知核验。

## 本站适合沉淀什么

- 通用学习方法和开发工具入门。
- 可公开讨论的竞赛、项目和科研准备经验。
- 对学校特定内容的待核验模板。
- 内容来源、核验状态和维护流程。

## 本站不适合沉淀什么

- 未经核验的学校政策和办事流程。
- 教师、学生个人联系方式或评价。
- 内部系统链接、账号、群号、学号、手机号。
- 受版权保护的课程材料全文。
