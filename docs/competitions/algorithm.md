# 算法竞赛入门

算法竞赛训练强调基本数据结构、算法思想、代码实现和复盘。初学阶段不要急于刷难题。

## 建议顺序

1. 熟悉输入输出和简单模拟。
2. 学习排序、二分、前缀和、双指针。
3. 学习栈、队列、堆、并查集等基础结构。
4. 做题后记录错误原因和边界条件。
