# 竞赛与实践总览

本页不说明学院或学校对竞赛的认定规则。竞赛认定、奖项统计、加分或报销等事项必须以正式通知为准。

## 可以沉淀的内容

- 通用训练方法。
- 公开平台和公开题目使用经验。
- 团队协作、代码管理和复盘方式。

## 谨慎处理的内容

- 校内选拔规则。
- 当前负责人和联系方式。
- 经费、认定、奖助相关流程。
