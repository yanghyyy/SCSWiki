# 两委会

学院目前通过**两委会**建立学生与老师之间的联系，具体分为**学生分会**和**团委秘书处**，由两委会与班级负责人选举出**主席团**成员。除去院级学生组织，一些**校级学生社团**也挂靠学院。

## 主席团

主席团成员全面统筹协调组织两委会和各班级负责人，进行学院活动的筹备、开展、执行。主席团成员主要分为两类：

- 学生分会主席（3人）
- 团委秘书处秘书长（1人）、团委秘书处副秘书长（1人）

## 学生分会

学生分会下设部门：纪检监察部、综合事务部、素质发展部、学习发展部、权益服务部。

### 纪检监察部

#### 部门简介

纪检监察部是学生两委会分管纪律和检查的重要职能部门，以服务学院学生为目的；以纪律检查、设施维护、一内部考核为主要工作，协助学院管理学生。纪检监察部是老师和学生沟通的桥梁，再学生管理中发挥了不可代替的作用纪检部旨在搞好两委会组织建设，为全员师生的工作、学习、生活营造良好的环境和氛围。

#### 部门工作

1. 寝室卫生与安全检查：定期开展寝室卫生检查及安全隐患排查（如大功率电器、违规电器使用），保障学生居住环境安全整洁；
2. 活动查人：负责学院各类活动的座签座次安排、签到与纪律监督，维护活动秩序；
3. 晚归点名：不定时抽查学生晚归情况，确保住宿安全；
4. 部门例会：每周召开例会，总结工作、部署任务，提升团队协作效率；
5. 内部考核：每周对学院学生违纪行为（如迟到、旷课、违规等）进行扣分统计与汇总，形成考核报告，在学院例会上进行通报，为学院评优评先提供数据支持；
6. 安全教育：联合学校保卫处定期开展消防安全演练，普及消防知识，提升学生应急处理能力。

#### 部门优势

1. 权威性与实践性并重：作为学院核心职能部门，成员可直接参与校园管理，积累宝贵的组织协调与问题处理经验；
2. 工作全面覆盖：从纪律监督到安全保障，从数据考核到实践演练，工作内容多元，助力成员全面发展；
3. 团队氛围融洽：部门注重传帮带文化，学长学姐耐心指导，定期开展团建活动，兼顾工作与成长；
4. 能力提升平台：通过策划活动、撰写报告、沟通协调等任务，锻炼逻辑思维、执行力与责任感，为未来职场奠定基础。

### 综合事务部

#### 部门简介

综合事务部原名大学生职业发展中心，简称大职发，现已史名成综合事务部。

#### 部门工作

1. 日常工作：每周干部大会的通知、点名、会议记录及汇总，部门内部例会，每周周记通知等；
2. 常规工作：各类值班表的制作、各类奖状的制作、新生档案的收集、优秀大学毕业生的材料审核、毕业证学位证盖章、整理毕业生档案等；
3. 承办活动：新生演讲比赛、优秀大学毕业生先进事迹报告会、职业生涯规划大赛等。

#### 部门优势

1. 我们享有更多与资深教师及行业导师亲密接触的机会，能够直接获得专业领域的深度指导与宝贵建议，为成员的成长铺设坚实基石；
2. 有机会与历届优秀成员进行面对面的经验分享，汲取他们的成功智慧与实战技巧，助力个人能力实现质的飞跃；
3. 我们注重成员全方位能力的培养，特别是办公自动化技能与高效工作方法的传授，确保每位成员都能在工作中游刃有余，展现专业风采；
4. 我们推动跨部门合作，通过项目共担锤炼团队协作，拓展人际网络。在这里，每位成员都能找到心灵契合的伙伴，综合事务部是一个温暖有凝聚力的集体。工作中细心严谨，软件操作娴熟，生活难题巧解，更有志同道合者携手同行。

### 素质发展部

#### 部门简介

西华师范大学计算机学院体育部隶属于学生会分会。2020年两委会改革，因此体育部更名为素质发展部，作为学生会的重要部门之一，以强健学生体魄弘扬体育精神为宗旨，以尽职尽责、服务同学的认真态度开展部门常规工作和特急活动，旨在增强同学们参加体育锻炼的意识，丰富同学们的课余文化生活，强健同学体魄，弘扬校园体育精神。

#### 部门工作

1. 策划组织各类体育赛事；
2. 对教学楼进行卫生检查；
3. 协助其他部门开展活动。

#### 部门优势

1. 可以第一时间知晓各类体育赛事的相关信息；
2. 可以提升自己的策划组织能力；
3. 可以亲自参与卫生检查工作；
4. 部门成员都特别好相处，部门内氛围十分融洽。

### 学习发展部

#### 部门简介

计算机学院学习发展部，是计算机学院两委会学生分会的一个分支。顾名思义，学习发展部主要分管与学习相关的工作，如学风建设和学术活动；部门日常工作是查早晚自习和查课，同时负责奖学金评选的检查与审核、各类准考证和证书的发放、学生证的补办等事宜。

#### 部门工作

1. 负责学院早晚自习的检查，查课；
2. 举行各类学术活动，如新生交流大会，三笔字大赛，英语词汇大赛等；
3. 各类准考证、证书的领取和发放，以及学生证的补办；
4. 奖学金评选的检查与审核。

#### 部门优势

部门有一名部长，四名副部长，部门内部友好和睦，有优秀的师哥师姐暖心陪伴， 在平时工作的同时还能与老师交流心得，获得更加全面学习生涯的指导。部门诚招积极认真，为人谦逊之豪杰，加入我们，你可以第一时间了解早自习的出勤情况和学院上课动态，同时还可以收获许多办公技能，还可以锻炼自己上台表达的能力，同时丰富自己的校园生活，收获更多朋友，快来加入我们吧。

### 权益服务部

#### 部门简介

计算机学院权益服务部是西华师大最温暖的“学生权益守护者”！作为学院与同学间的桥梁，我们始终秉持“权系你我，益在大家”的宗旨，用行动为同学们的学习生活保驾护航。这里不仅是服务同学的平台，更是锻炼能力、收获成长的大家庭。

#### 部门工作

1. 绿色通道：为新生开通入学无忧通道；
2. 奖助体系：精准完成奖学金评定与材料审核；
3. 贷款服务：协助办理生源地贷款全流程；
4. 贫困认定：科学开展家庭经济困难学生认定；
5. 育人活动：策划“资助育人”系列主题活动；
6. 卫生共管：与素拓部联合打造洁净校园。

#### 部门优势

1. 能力提升：组织协调/沟通谈判/问题解决能力全方面进阶；
2. 成长机遇：学生干部经历为评优评奖强势加分；
3. 人脉拓展：结识各专业优秀伙伴，建立跨年级社交圈；
4. 思维升级：学会系统化分析问题，培养全局视角；
5. 温暖团队：收获一群志同道合的战友，让大学充满归属感；
6. 自我突破：在服务中实现德智体美劳全面发展。

## 团委秘书处

团委秘书处下设部门：创新创业部、青年志愿者协会、文艺部、宣传部、组织部、新媒体中心、大学生艺术团、党建办。

### 创新创业部

#### 部门简介

计算机学院创新创业部，作为学院两委会的核心部门之一，隶属于团委。我们始终以服务全院同学的 “学术、文化、科技、创新、创业、就业” 需求为使命，既是各类学科专业竞赛与创新创业赛事的组织者与推动者，也是校园学术文化氛围的重要建设者。在这里，我们不仅协助学校举办ACM、蓝桥杯、计算机设计大赛等专业高水平竞赛，更全面组织互联网+、挑战杯、大创赛等创新创业赛事，为同学们提供展示才华、实践创新的舞台。同时，我们也注重部门成员的各方面的成长，打造一个集学习、实践、交流、成长的良好环境。

#### 部门工作

1. 学术文化推广：在团委、学生会的指导和安排下，围绕同学们的学习与成长需求，策划并开展学习竞赛、专题讲座、学术论坛等多样化活动，活跃校园第二课堂，助力同学们营造浓厚的学术氛围；
2. 创新赛事引领：全面负责 ACM、蓝桥杯等计算机类专业竞赛的组织与管理，协助科研立项、创新创业项目等相关工作；同时，承办互联网+、创青春、挑战杯等重量级创新创业大赛，帮助同学们在实践中积累经验，开拓视野；
3. 技能培训与协助工作：部门将不定期为成员提供技能培训，涵盖 PS、AE、PR、Office办公等常用软件，提升实用技能和工作效率；同时负责管理学院的音响设备，协助其他部门顺利开展活动，并为其他部门提供技术与设备支持。

#### 部门优势

1. 前沿赛事接触，助力未来发展：加入创新创业部，你将近距离接触各类竞赛，提前掌握赛事流程与经验，为未来参赛与就业积累宝贵优势；
2. 技能提升，全面成长：拥有部门培训 + 实践机会的双重成长。在这里，你既能获得软件与技术上的硬实力提升，也能在团队协作与项目执行中锻炼组织能力和沟通能力；
3. 人脉拓展，收获友情：在创新创业部，你会遇到一群志同道合、积极进取的小伙伴，部门成员之间也会互相分享各类学习资料。部门还会与其他部门联谊，更加丰富你的大学生活，打开交友与合作的新天地。

### 青年志愿者协会

#### 部门简介

“青志协”，即青年志愿者协会。我们部门始终秉持“奉献、友爱、互助、进步”的志愿精神，致力于在学院内助力同学们理解志愿精神，熟知志愿事迹，积极参与志愿活动，从而培养大家对公益志愿活动的浓厚热爱。倘若您热衷志愿活动，心怀助人之念，赶快加入我们吧。

#### 部门工作

我们部门涵盖两个大类的工作，青志协和心咨部。

**青志协方面**：

1. 我部门主要负责“三下乡”“春晖行动”等志愿实践活动的展开以及后续工作材料收集整理；
2. 感谢恩师·你我同行，等大型公益活动的志愿直播系列活动；
3. “图书漂流”活动，让各位在书籍的海洋里面遨游；
4. 学院不定期举办的志愿活动，如去幼儿园进行公益讲堂，献血志愿者的招募等等。

**心咨部方面**：

1. 这是一个面向学生开设的心理咨询窗口，积极开展各项与心理健康相关的活动，给我院师生提供心理辅导与帮助；
2. 针对生活心理问题，适时开展团体心理辅导；
3. 普及心理健康相关知识，促进学生身心全面发展，健康成长；
4. 开展许多心理相关活动，如三月女生节活动、“5.25”“12.5”特色心理活动，以及不定时开展的心理观影活动和心理情景剧等。

#### 部门优势

部门会不定时团建，充满欢声笑语，留下大学美好快乐的回忆，收获一群志同道合的伙伴。部门的师兄师姐们也会热心的解答问题，无论是生活上的还是学习上的，都会互帮互助。同时我部门特色——青志协入会仪式，部门每年都会为新来的干事举行入会仪式。

### 文艺部

#### 部门简介

计算机文艺部，是计算机两委会团委秘书处的一个分支。文艺部作为学生会的一个常设部门，承担着丰富学生课余文化生活、通过策划和组织各类文艺活动，文艺部为大家提供了一个展示自我、锻炼能力的平台。文艺部还需与其他部门共同协作，共同推动校园文化的繁荣发展。

#### 部门工作

1. 三好标兵擂台赛：为三好标兵擂台赛的举办做好幕后工作，进行节目筛选以及节目的顺序的制定；
2. 少数民族联谊会：为发扬各民族习俗以及文化，我们进行物资采购布置现场等工作并为参与者介绍相应民族文化；
3. 学院毕业晚会：与各部门一起为毕业晚会的召开进行幕后工作；
4. 每周的例行会议：每周参与学院两委会会议并且开展部门会议；
5. 各部门间联合工作：院内部门之间会经常进行合作开展活动。

#### 部门优势

1. 提升综合能力在活动策划、新媒体运营等工作中，锻炼组织协调、文案创作、沟通谈判等实用技能，增强职场竞争力；
2. 拓展优质人脉结识跨专业伙伴与校内外前辈，构建互助网络，为学业、创业或职业发展提供支持；
3. 积累实践经验通过组织活动、商务合作等经历，积累项目管理、谈判等经验，丰富履历，助力求职；
4. 深化自我认知在部门分工中发现自身优劣势，明确适配的发展方向，平衡个人与团队关系。

### 宣传部

#### 部门简介

用笔尖记录美好，用镜头定格时光，用设计传递想法。你的大学生活，不应只有学习和宿舍，还应有一群志同道合的伙伴和一个展现自我的平台。你的选择绝不会错，你的未来将由你书写！宣传部，等你来定义。

#### 部门工作

一．活动影像记录（拍摄活动照片）

1. 全流程跟拍：覆盖活动筹备、进行、收尾各阶段，包括场地布置、嘉宾签到、精彩互动、颁奖仪式等关键节点；
2. 多视角拍摄：运用广角镜头捕捉全景，特写镜头记录细节，结合低角度、俯拍等创意构图增强画面感染力；
3. 即时素材筛选：现场快速筛选优质照片，通过云端共享实时反馈至新媒体团队，用于活动直播或预热宣传。

**二. 新闻稿件创作（撰写活动稿件）深度报道**：呈现活动全貌，穿插背景延伸、嘉宾金句、观众反响等多元视角。

三. 空间视觉营造（活动黑板报设计）

1. 主题策划：结合节日节点（如校庆、教师节）或社会热点（如环保、航天）确定创作方向；
2. 立体化呈现：融合粉笔画、彩纸拼贴、立体浮雕等技法，搭配LED灯带、二维码互动等数字元素。

#### 部门优势

1. 学时易拿：参加活动多，各种类型的学时都能拿到；
2. 发展兴趣爱好：部门内有两台相机供大家练习使用；

组织部是团委的核心职能部门之一，主要负责本学院团委相关的工作。我们以“严谨、务实、创新”为工作准则，同时注重成员个人能力的培养和团队协作能力的提升，加入我们，一起将团组织打造成联系青年、服务青年的坚强堡垒，让组织部成为我们温暖的大家庭。

#### 部门工作

### 新媒体中心

#### 部门简介

计算机学院新媒体中心是计算机学院两委会团委秘书处的一个重要分支，专注于新媒体的运营和管理。中心主要负责“西华师范大学计算机学院”官方公众号和易班平台的运营，以及活动发布和学时发放相关工作。我们的使命是在学院内外建立有效的信息交流平台，以提升学院的形象和影响力。

#### 部门工作

1. 运营学院官方账号：负责“西华师范大学计算机学院”公众号的日常维护与推文发布，定期分享保研资讯、学术活动及各类重要信息；
2. 易班工作：管理和运营易班平台，发布计算机学院相关话题，组织承办“易班嘉年华”及其他线上线下活动，以促进同学之间的互动与交流；
3. 第二课堂管理：审核并发布“第二课堂”活动，负责相关学时的发放工作，确保每位参与者获得相应的学时认证；
4. A级证书认定：负责四川省大学生“A级证书”认定的相关工作，帮助同学获取省级认证，增强其综合素质与竞争力。

#### 部门优势

1. 氛围温馨，归属感强：以 “一家人” 为理念，无层级隔阂，成员互帮互助，定期组织团建与分享会，能快速融入集体；
2. 学长学姐贴心指导：经验丰富的学长学姐主动帮新成员解决工作（如公众号运营、学时处理）与生活疑问，手把手教学；
3. 工作轻松，学工平衡：合理安排任务，不占用过多学习休息时间，灵活协作，实现学习与实践两不误；
4. 实战练技能，促个人提升：可实操秀米排版、易班运营等工作，快速掌握新媒体技能，积累实践经验。

### 大学生艺术团

大学生艺术团下设四个部门，分别是：主持队、辩论队、礼仪队、舞蹈队。

> 主持队

#### 主持队 部门简介

舞台之上，他们是点亮灵魂的话语艺术家加入主持队，用声音传递炽热情感，以语言编织斑斓梦想。从校园晚会到各类活动，我们执笔主持稿、登台控全场，合力打造专属视听盛宴，让每一次登台，都成为难忘的高光时刻。

#### 主持队 部门工作

为了登台时的专业闪耀，日常训练聚焦仪态管理、发声技巧、情感表达，全方位打磨主持功底。学院活动主持重任由我们担当，每周训练中选拔主持人，从稿件撰写到现场排练，全程参与，让每一场活动都因我们的主持而精彩。

#### 主持队 部门优势

1. 掌握公众演讲必杀技，告别社恐、自信开麦；
2. 练就字正腔圆普通话，播音级语言质感拿捏；
3. 解锁朗诵、辩论隐藏技能，表达力 upup；
4. 收获超强团队默契，和同频伙伴并肩战斗，攒下珍贵友谊。

> 辩论队

#### 辩论队 部门简介

我们以辩为舟，穿梭于观点的星河，在 “人性善恶” 的探讨里洞察幽微，于 “时代浪潮” 的争辩中把握脉搏。我们用语言编织思辨的锦缎，让每一次交锋，都成为点亮认知的星火，让热爱思辨的你，找到灵魂共鸣的同路人，共赴思维探险之旅。

#### 辩论队 部门工作

开展逻辑训练、辩题剖析、模拟辩论，提升辩论能力，最终选出种子选手代表学院参加西华之声。

#### 辩论队 部门优势

在这里，能沉浸式感受观点交锋的趣味，锻炼出惊艳的口才，不再怯场；在这里，能轻松获取第二课堂学时，没有繁冗的工作任务；在这里，队友们同频又合拍，相处氛围融洽温暖，既能在思辨中成长，也能收获真挚友谊。

> 礼仪队

#### 礼仪队 部门简介

礼仪队作为展现组织形象的重要窗口，以专业的礼仪服务为核心，通过系统培训让成员掌握规范的礼仪技巧，在各类正式活动中承担引导、接待、颁奖等工作，既保障活动有序开展，也传递出尊重得体的社交态度，是提升活动规格与组织形象的重要力量。

#### 礼仪队 部门工作

1. 负责学院内各类正式活动的礼仪服务，如开幕式、闭幕式、颁奖典礼、毕业晚会等，具体工作包括引导嘉宾、颁奖递物等工作；
2. 成员需经过系统的礼仪培训，掌握基本的站姿、走姿、微笑、手势等礼仪规范，以及接待、沟通等技巧。

#### 礼仪队 部门优势

加入礼仪队，既能通过系统的礼仪培训塑造优雅气质，在各类重要活动中锻炼沟通、应变与团队协作能力，积累丰富的社交经验和视野；又能在为活动与组织增光添彩中收获成就感，增强自信心，实现个人素养与价值感的双重提升。

> 舞蹈队

#### 舞蹈队 部门简介

舞蹈队是校园文化里的闪亮之星，为大学生活注入无限活力与魅力。这里汇聚了一群怀揣舞蹈梦想、热爱跳舞的同学。在忙碌的学业之余，舞蹈队为大家提供了释放压力的空间，让同学们尽情享受舞蹈带来的快乐与成就感。无论你基础如何，只要对舞蹈有兴趣，舞蹈队都热忱欢迎你的加入！舞蹈队主要涵盖 JAZZ、KPOP、民族舞等多种风格。

#### 舞蹈队 部门工作

1. 开展舞蹈基本功训练；
2. 进行成品舞教学与走位排练；
3. 参与活动彩排及正式表演。

#### 舞蹈队 部门优势

舞蹈是独特的艺术表达形式，队员们通过不同舞蹈作品传递细腻情感、展现深刻思想，收获美的享受。在学习与表演过程中，不断提升审美能力，更好地感知美、创造美。提供社交平台，结识志同道合的朋友，收获自信心和满足感。
