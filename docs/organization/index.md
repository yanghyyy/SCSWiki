# 学院组织架构

本章节所有信息全部来源于西华师范大学计算机学院官网，如有冲突请以官网信息为准！

## 学院简介

西华师范大学计算机学院成立于1993年，成立时名称为四川师范学院计算机系，1996年更名为四川师范学院计算机科学系，2003年四川师范学院更名为西华师范大学，我院随之更名为西华师范大学计算机学院。

学院拥有一支思想素质好、学术水平高、教学经验丰富的师资队伍，现有教职员工58人，正高级9人，副高级14人，具有博士学位教师16人（含外籍教师1人）。学院在教学科研方面成果显著，获得国家科技进步三等奖1项，省（部）级科技进步二等奖3项、三等奖2项；获省级教学成果二等奖2项、三等奖2项；获得四川省自然科学奖三等奖1项；获中国医药教育协会科技创新基础研究三等奖1项；教职工主持和参与国家级、省部级项目40余项，编著教材20余部。2015年学院党委被中共四川省委教育工作委员会授予“四川省高等学校优秀服务型党组织”荣誉称号。

学院拥有软件工程一级学科学术硕士学位授予权、电子信息（计算机技术和人工智能方向）专业硕士学位授予权，现有计算机科学与技术（师范）、网络空间安全、软件工程3个本科专业，软件工程专业招收本科、研究生国际留学生，在校全日制本科生1777人（含留学生110人），在校硕士研究生135人（含留学生7人），计算机科学与技术（师范）专业2001年被列为四川省本科人才培养基地、2008年被评为四川省特色专业、2019年获批四川省一流专业、2024年通过普通高等学校师范类专业第二级认证；计算机科学与技术教学团队被评为四川省优秀教学团队；软件工程专业2012年被授予四川省首批“卓越工程师”培养计划、2016年评为四川省应用型示范型专业和四川省教育综合改革试点专业；软件工程一级学科硕士点，2024年通过国务院学位办学位点专项合格评估。

学院设有计算机科学与技术、软件工程、网络空间安全3个教研室；设有物联网感知与大数据分析南充市重点实验室和校级人工智能研究所，“图像智能分析创新团队”校级科研团队；有软件测试、模电和数电、机器人、计算机组成原理、网络攻防、嵌入式系统及物联网等各类专业实验室，拥有各类仪器设备总价值2000余万元。

学院禀承“勤奋、求实、敬业、创新”的校训，以“强理论知识、重实践技能、育工科人才”的能力导向型创新人才培养理念，推进人才培养模式改革，对标师范认证和工程认证标准，深化教学改革和创新实践训练，人才培养质量显著提升。在蓝桥杯全国软件和信息技术专业人才大赛、中国大学生计算机设计大赛、四川省大学生机器人大赛、“挑战杯”中国大学生创业计划大赛、四川省“互联网+”大学生创新创业大赛等各类比赛竞赛中多次获得国家、省级奖项。2016年以来，在人才培养方面3次获评西华师范大学学生工作评估“一等奖”，3次获评西华师范大学毕业生就业工作“先进集体”，5次获评西华师范大学创新创业工作“先进集体”，4次获评西华师范大学考研工作“先进集体”。近3年毕业生就业率保持在90%以上，部分毕业生考取公务员、事业单位，入职腾讯、字节跳动、中国电信、中国联通、中国移动等各大科技公司。

欢迎广大立志从事计算机教育和IT行业的青年报考计算机学院，学院必定为您开启美好的明天！

## 专业介绍

### 计算机科学与技术（师范）

修业年限：学制四年，弹性修业年限3-6年。

授予学位：工学学士学位

招生类别：理工

培养目标：本专业培养具有良好的思想政治素质，较系统地掌握计算机科学的基本理论、基本技能与基本方法，富有创新精神，能适应基础教育改革发展需要的中小学信息技术教师、教学研究人员及教育行政管理人才和其它高素质专门人才。

主干课程：数据结构、计算机组成原理、面向对象程序设计、计算机操作系统、数据库系统原理、计算机网络原理、教师口语、信息技术学科教学论、课程设计与评价、教育心理学、教育学等。

升学及就业前景：学生毕业后可在企、事业单位从事信息技术相关工作，也可在各类各级学校从事教学、科研及管理等工作，也可继续攻读本专业和相关专业硕士研究生。

### 软件工程

修业年限：学制四年，弹性修业年限3-6年。

授予学位：工学学士学位

招生类别：理工

培养目标：培养满足企业软件开发需求的知识、能力、素质全面发展，具有高尚职业道德和社会责任感，工程实践能力强，系统掌握软件工程学科的专业基础知识、基本方法和技能，具有良好软件工程素养及创新意识的高级软件工程技术人才。

主干课程：主干课程：电子技术基础、计算机组成原理、操作系统、数据结构、计算机网络、数据库系统、软件项目管理、算法设计与分析、程序设计语言、面向对象程序设计、软件测试、机器学习、大数据技术、物联网应用、机器视觉、嵌入式系统等

升学及就业前景：学生毕业后能够在企事业单位从事软件开发及相关技术工作，也可在中等和高等学校从事软件工程专业相关的教学工作，也可继续攻读本专业和相关专业硕士研究生。

### 网络空间安全

修业年限：学制四年，弹性修业年限3-6年。

授予学位：工学学士学位

招生类别：理工

培养目标：培养素质、知识、能力全面发展，具有深厚的人文底蕴、扎实的网络空间安全专业知识、强烈的创新意识、宽广的国际视野，具备网络空间安全科学研究、技术开发和应用服务工作能力以及良好网络空间安全素养的工程型人才。

主干课程：网络空间安全导论、计算机网络原理、密码学基础、网络攻防技术、网络安全技术、实用网络技术、安全协议分析、信息安全综合实训、信息系统安全及测评技术、网络管理与维护、综合实践课程等。

升学及就业前景：能够在网络空间安全产业以及其他国民经济部门从事安全运维、网络渗透、网络空间安全规划管理、信息安全等级保护、安全产品开发等工作，也可继续攻读本专业和相关专业硕士研究生。
