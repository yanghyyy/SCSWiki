# 学生社团

目前学生社团指的是西华师范大学社团管理中心注册的学生社团，这里只介绍挂靠西华师范大学计算机学院的学生社团。

## 程序设计算法协会

### 简介

西华师范大学程序设计算法协会（以下简称 “协会”）成立于2023年，是计算机学院指导下的专业学生社团，由副院长潘伟、周坤老师担任指导老师。协会以 “深耕算法技术、培育创新人才” 为宗旨，始终立足学校人才培养目标，致力于为全校程序设计与算法爱好者搭建专业化、体系化的交流成长平台。协会以系统化培训为根基、赛事实践为抓手、团队协作为支撑，着力培养兼具扎实技术功底、突出创新思维与优秀协作能力的复合型计算机人才，是学校算法竞技人才孵化与计算机学科后备力量培养的核心阵地。

### 核心工作

协会围绕人才培养核心目标，构建了“培训-竞技-成长”全链条工作体系，核心工作板块如下：

- 赛事体系搭建与组织：负责全流程举办校内程序设计系列赛事，包括西华师范大学程序设计竞赛、程序设计竞赛个人排位赛、团体程序设计天梯赛校内选拔赛及蓝桥杯全国软件和信息技术专业人才大赛校内赛，打造常态化、阶梯式的竞技舞台，营造浓厚的算法学习与竞赛氛围，搭建校级赛事与省级、国家级高等级赛事的衔接通道。
- 人才选拔与集训培养：统筹省级、国家级程序设计竞赛的报名、选拔与备赛工作，专项组建XCPC集训队，开展全周期系统性训练。通过常态化模拟赛、核心算法专题研讨、赛事复盘总结、跨校交流拉练等多元形式，全方位提升队员竞技水平，助力优秀学子冲击更高规格的赛事舞台，斩获优质竞赛成果。
- 技术普及与能力赋能：定期开展全覆盖式算法培训、行业技术讲座与开源项目实践活动，内容覆盖数据结构、动态规划、图论、算法工程化等核心知识与实战技能，构建从入门到进阶的全层级培训体系，帮助会员完成从理论知识到实战能力的转化。同时通过常态化技术研讨会、代码开源分享、项目协作开发等形式，搭建自由开放的技术交流生态，激发会员创新活力。
- 社团规范运营与品牌建设：建立健全协会规章制度与运营体系，统筹社团日常管理、会员服务、文化建设与对外交流工作，打造辨识度高、影响力强的校园专业社团品牌，持续吸纳优质人才，推动协会高质量可持续发展。

### 协会成果

成立至今，协会始终坚持以赛促学、以赛促练、以赛育人，带领广大会员深耕算法领域、勇攀竞技高峰，成员在蓝桥杯全国软件和信息技术专业人才大赛、XCPC系列赛事等国内重量级程序设计竞赛中屡创佳绩，累计斩获省级及以上奖项百余项，用硬核实力印证了协会的育人底色与培养成效。
